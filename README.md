# Autonomous AI Engineer

An autonomous agent that generates, tests, and deploys full-stack web applications and Python scripts based on user prompts.

## Features

- **Autonomous Code Generation**: Generates Python scripts and HTML/CSS/JS web pages using AI.
- **Automated Testing**: Runs tests on the generated code to ensure functionality.
- **Instant Deployment**: Deploys the code and provides a live preview.
- **Full-Stack Support**: Handles both backend logic and frontend templates.
- **Web Interface**: A clean, modern UI to interact with the AI engineer.

## Technical Stack

- **Backend**: Flask (Python)
- **Frontend**: HTML5, CSS3, JavaScript
- **AI Integration**: Gemini / LiteLLM (via agents)

## Project Structure

- `agents/`: AI agents for generation, testing, and deployment.
- `templates/`: HTML templates for the web interface.
- `static/`: CSS and static assets.
- `generated_code/`: Storage for code generated by the AI.
- `deployed_apps/`: Location for deployed applications.

## Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/Moksha-132/Autonomous-AI-Engineer.git
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Set up your environment variables in a `.env` file.
4. Run the application:
   ```bash
   python app.py
   ```
